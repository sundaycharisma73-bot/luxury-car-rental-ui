# Luxury Car Rental UI

Static HTML & CSS front-end for the Luxury Car Rental website.

This repository contains a static website (HTML, CSS, and assets). It is intended to be served as a static site (no Node runtime required).

Live demo: (add your demo URL)

## Quick start

1. Clone the repo:
   git clone https://github.com/sundaycharisma73-bot/luxury-car-rental-ui.git
   cd luxury-car-rental-ui

2. Open locally
   - Open `index.html` in your browser, or run a simple static server (recommended):
     - With Node (http-server): `npx http-server -c-1` then open http://localhost:8080
     - With Python 3: `python3 -m http.server 8000` then open http://localhost:8000

3. Edit files
   - HTML files are in the repository root (or subfolders). Edit and refresh the browser.

4. Deployment
   See docs/SETUP.md for deployment options (GitHub Pages, Netlify, Vercel, Docker).
